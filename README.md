# Exercícios Java Collections

## Sobre

Esse repositório tem exercícios que estou fazendo para aprender Java Collections.

Aqui eu treino coisas como:
- List
- Set
- Map
- Comparable e Comparator (ordenação)
- Generics (tipos genéricos)
- Adicionar, remover e buscar dados

---

## Estrutura

Os arquivos estão organizados assim:

src/Main/java/

- comparableXComparator
- generics
- list
- set
- map

Dentro dessas pastas tem exercícios separados por tipo (básico, pesquisa, ordenação).

---

## O que precisa

- Java instalado (JDK 17 ou mais novo)
- Pode usar uma IDE ou rodar pelo terminal

---


Clonar o repositório:

```
git clone https://github.com/BRYANGabb/Java-Collections.git
```

Entrar na pasta:

```
cd Java-Collections
```

Compilar:

```
javac -d bin src/Main/java/**/*.java
```

Rodar um exercício:

```
java -cp bin Main.java.map.OperacoesBasicas.NomeDaClasse
```

(Substitua NomeDaClasse pela classe que deseja rodar)
